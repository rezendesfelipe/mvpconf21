# Remote State Complexo
## Introdução
Este cenário apresenta um uso 'mais completo' do State do terraform, porque está armazenando o terraform em uma storage account. Além disso, informações sensível do ambiente não são expostas graças ao uso da feature de **partial configuration**. [referência](https://www.terraform.io/docs/language/settings/backends/configuration.html#partial-configuration)
